1: Co je JavaScript a k jakým účelům se používá?

Doplň vhodné odpovědi a využij k tomu nečíslovaného seznamu.

Použij v textu aspoň 3 internetové odkazy na zajímavé příklady využití JS!

JavaScript je multiplatformní, objektově orientovaný, událostmi řízený skriptovací jazyk, jehož autorem je Brendan Eich z tehdejší společnosti Netscape. Jeho syntaxe (zápis zdrojového textu) patří do rodiny jazyků C/C++/Java, ale JavaScript je od těchto jazyků zásadně odlišný sémanticky (funkčně, pricipiálně), jde o jiný jazyk.

2: Jaká pravidla platí pro psaní kódu v JavaScriptu?

Napiš aspoň pět zásad ve podobě číslovaného seznamu.

Použij angličtinu!

  1. Always end a simple statement with a semicolon.
  2. Use spaces liberally throughout your code.
  3. Don't forget to use camelCase.
  4. JavaScript doesn't care about single or double quotation marks.
  5. Variables must be defined before being used.

6: Která popup okna lze využít v JavaScriptu?

K uvedeným příkladům připište do poznámky, co se stane na webové stránce.

Na konci doplňte aspoň dvě varování, proč bychom se měli vyhýbat použití popup oken v reálných webových aplikacích.

alert()

/* Upozorní uživatele popup oknem na špatně zadané heslo */
alert('Nezadal jsi správné heslo!');

prompt()

/* Do proměnné password vloží heslo zadané uživatelem */
let password = prompt("Heslo:");

confirm()

/* Zeptá se uživatele, jestli chce přejít na stránku Seznam.cz (nabídne dvě možnosti - OK a Cancel) */
if (confirm('Chceš přejít na stránky Seznam.cz?')) location.href='https://seznam.cz';
Varování
  1. Jde o zastaralý způsob zobrazení dodatečných informací.
  2. Většina prohlížečů popup okna blokuje z bezpečnostních důvodů.
